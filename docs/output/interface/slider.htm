Slider

A vertical or horizontal slider. Sliders are a type of Widget and inherit its methods and properties.

Example:

a = Slider()
a.setValue( .5 )

a.isVertical = false

Properties

  • isVertical : Boolean. Default : true. Whether the slider is horizontal or vertical.
  • value : Float. Default range: { 0, 1 }. Default: 0.
