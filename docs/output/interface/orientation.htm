Orientation

When used in sketches running on mobile devices, this widget will use the orientation of the device as a signal.

Example:

pwm = PWM()
o = Orientation()

pwm.frequency = o.X
pwm.amp = o.Y
pwm.pulsewidth = o.Z

Properties

  • x : Float. Rotation of the device along the x-axis.
  • y : Float. Rotation of the device along the y-axis.
  • z : Float. Rotation of the device along the z-axis.

Methods

  • start : Begin querying sensors for orientation data. Note: this is called automatically when Orientation is used with Gibber's mapping abstractions.

  • stop : End querying sensors for orientation data.

See the Widget prototype for relevant methods.
