Crossfader

A horizontal crossfader. Crossfader are a type of Widget and inherit its methods and properties.

Example:

// crossfade between two drum loops
crossfader = Crossfader()

loop1 = Drums('x*o*x*o-')
loop2 = Drums('x*ox*xo-').pitch(2)

loop1.amp = loop2.amp = crossfader

loop2.Amp.invert()

Properties

See the Widget prototype for many other relevant properties.

  • value : Float. Default range: { 0, 1 }. Default: 0.
