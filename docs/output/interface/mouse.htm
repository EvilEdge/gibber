Mouse

This object presents signals derived from the position of the mouse cursor in the browser window.

Example:

a = Mono().play( Rndi(0,12), 1/4 )

a.cutoff = Mouse.X
a.resonance = Mouse.Y

Properties

  • x : Float. A value between 0 and 1 representing the x position of the mouse in the browser window.
  • y : Float. A value between 0 and 1 representing the y position of the mouse in the browser window.

Methods

  • on : Begin polling mouse position data.
  • off : Stop polling mouse position data.
