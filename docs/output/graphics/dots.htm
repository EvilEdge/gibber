Dots

A post-processing shader recreating the halftone technique (http://en.wikipedia.org/wiki/Halftone)

Example:

a = Cube()
a.scale = 2
a.spin( .05 )
b = Dots({ scale:.25 })

Properties

  • scale : Float. The size of the dots. Larger values result in smaller dots.
  • center : THREE.Vector2. Center position of dots
  • angle : Float. Angle of dots in radians
